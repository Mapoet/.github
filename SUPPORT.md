# 支持与协助 / Support & Assistance

Mapoet 致力于为所有贡献者和使用者提供清晰、及时的帮助。此文档适用于任何继承 `Mapoet/.github` 模板的仓库。

## 快速入口 / Quick Links

- 📚 常见问题：请查看各项目仓库的 `README` 与 `docs/` 目录。
- 🧭 贡献指南：参阅 [CONTRIBUTING.md](./CONTRIBUTING.md)。
- 🤝 行为准则：参阅 [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)。
- 🔒 安全问题：请按照 [SECURITY.md](./SECURITY.md) 的流程私下报告。

## 获取帮助 / Getting Help

1. **一般使用问题**：在相关仓库的 Discussions 或 Issue 中提问，并描述上下文与复现步骤。
2. **模板或流程疑问**：在本仓库开启 Issue，说明你遇到的问题、期望的行为以及影响的仓库。
3. **紧急支持**：如遇到阻塞性问题，可发送邮件至 `support@mapoet.dev`，我们会在 2 个工作日内回复。

## 反馈建议 / Feedback

- 任何关于模板、流程或社区体验的建议都欢迎通过 Issue、Discussions 或 PR 提出。
- 对于重大变更，请附上影响评估与迁移建议，帮助维护者协调后续工作。

感谢你参与 Mapoet 社区，让我们的生态不断进步。
